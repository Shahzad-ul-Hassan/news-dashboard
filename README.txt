README (Quick Start)

This is a static dashboard (HTML/CSS/JS). No backend required.

A) Open locally
1) Extract the ZIP
2) Open: news-dashboard/index.html

B) Publish on GitHub Pages
1) Create a GitHub repository
2) Upload ALL files/folders from the ZIP to repo ROOT (top level)
3) Settings -> Pages
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)

LOCKS
- No buy/sell signals, no predictions, no trading advice.
- All times are 12-hour AM/PM.
- DST handled by browser time zones (Europe/London, America/New_York, etc).
